# GrowGreen

## 프로젝트 소개

- 습관을 개선하여 건강한 삶을 살아가는 것을 희망하는 사람들을 위한 사이트입니다.
- 실생활에서도 간단하게 실천할 수 있는 식단과 운동 및 건강관리 팁 등을 제공합니다.
